# CardBoard

クレジットカードの引き落とし管理アプリ。

## これは何？

**直近2ヶ月の収支を把握するためのツール**です。

口座残高・固定収支・カード利用額を入力すると、引き落とし後の予測残高をリアルタイムで確認できます。長期的な支出の集計や履歴管理を目的としたアプリではありません。データはブラウザのlocalStorageにのみ保存されます。

## 主な機能

- 引き落とし後の予測残高の計算
- カード別・月別の利用額入力
- 引き落とし日・締め日のカレンダー表示（祝日・振替休日対応）
- 固定項目（給与・積立など）の管理
- カード情報（ブランド・年会費・問い合わせ先）の管理

## 技術

単一HTMLファイル。外部依存はChart.jsとGoogle Fontsのみ。フレームワーク不使用。
